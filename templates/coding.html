Coding / Dev / Programming

Okay, so it is definitely a bit pretentious/misleading of me to have a whole page dedicated to this as it implies that I'm more experienced and prolific than I am!

I recently got into programming/scripting (I mean, literally three or so months ago) and quite enjoy it, so I guess this page is here because I see it playing a bigger part in my various activities from here on.

Languages

I've mainly been working on Python, but am also learning C. Python is very current and seems a good place to start. I wanted to get familiar with C because so many languages (Java, C++, C#) are derived from it. Also, so much stuff is still written in C.

Current projects...

Editing SCORE files

The big aim with learning a bit of programming is to improve my workflow in SCORE, more specifically – to supercede my Editscor script for editing slurs/ties with a more intelligent and flexible program. With some helpful hints from a friend, I'm reading the binary data successfully. I just have to begin the long job of building a scalable program!

This is being done in Python, though I may move over to C.

Webscraping

As part of my job at Samling, we have to edit a set of artist pages (350 individual pages!) one at a time. There's no database or way for us to do global edits across the 350 pages. Often it takes ages to just go through every page and establish what information they do and don't have. So I wrote a Python script to do this for me, which summarises which pages have what information. It also spots inconsistencies. I then just make the desired edits to the relevant pages without the time-consuming process of checking each page for a specific element.

Scheduling

This has taken a bit of a backburner but I'll pick it up when I can... Again, at Samling I spend so much time sorting artists into groups and making schedules. I am working on a way to automate this – particularly the group sorting, which can often take the best part of a day.

Other stuff

This website!

I guess this website constitutes as part of my ongoing learning of Python. It is built using Python with the Flask framework and Jinja template engine. Locally, there are some small JSON databases. I then use Frozen-Flask to 'freeze' the site into a series of static html sheets, which I just upload to my VPS.

Git

I'm a massive fan of Git and am spending time getting to know its awesome capabilities. I do have a Github account, but prefer self-hosting wherever possible – so I have my own Git server on this VPS. I will eventually add a web interface so this is publically browsable, most likely using Stagit.
