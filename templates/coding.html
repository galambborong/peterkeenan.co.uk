Coding / Dev / Programming

Okay, so it is definitely a bit pretentious/misleading of me to have a whole page dedicated to this as it implies that I'm more experienced and prolific than I am.

I recently got into programming (I mean, literally three or so months ago) and quite enjoy it, so this page is here because I see it playing a bigger part in my various activities from here on.

Languages / paradigms

I started learning Python in April and am enjoying this a lot. It was a good introduction to object-oriented, imperative programming. I've recently started learning JavaScript and feel the principles I grasped through Python are translating well (so far, at least!).

I am also dabbling with Haskell. For whatever reason, I find functional programming appealing and I find the Haskell language elegant and powerful.

Current projects

Editing SCORE files

The big aim with learning a bit of programming is to improve my workflow in SCORE, more specifically – to supercede my Editscor script for editing slurs/ties with a more intelligent and flexible program. With some helpful hints from a friend, I'm reading the binary data successfully. I just have to begin the long job of building a scalable program!

Webscraping

Samling Artist pages

As part of my job at Samling, we have to edit a set of artist pages (350 individual pages!) one at a time. There's no database or way for us to do global edits across the 350 pages. Often it takes ages to just go through every page and establish what information they do and don't have. So I wrote a Python script to do this for me, which summarises which pages have what information. It also spots inconsistencies. I then just make the desired edits to the relevant pages without the time-consuming process of checking each page for a specific element.

Haskell project

I have started a project in Haskell which scrapes a website for data using APIs and such. It's early days, but I'll hopefully get it on my GitHub page soon.

Scheduling

This has taken a bit of a backburner but I'll pick it up when I can... Again, at Samling I spend so much time sorting artists into groups and making schedules. I am working on a way to automate this – particularly the group sorting, which can often take the best part of a day.

Other stuff

This website!

I guess this website constitutes as part of my ongoing learning of Python. It is built using Python with the Flask framework and Jinja template engine. Locally, there are some small JSON databases. I then use Frozen-Flask to 'freeze' the site into a series of static html sheets, which I just upload to my VPS. This has the advantage of being minimal/static but immediately scalable if/when I need to move to a dynamic site.

Though minimal and simple is what I'm going for, this site is lacking in certain aspects – mainly responsiveness and capitalising on HTML5. Both of these will be addressed in due course.

Git

I'm a massive fan of Git and have spent time learning more about its lower-level functions (commonly referred to as its "plumbing"). I host my own Git server on this VPS (although its web interface is a bit careless at present, I have not yet set up the post-hooks). I am on GitHub too.
